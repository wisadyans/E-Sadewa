Teknologi yang Digunakan

Dalam proyek SadewaTech, beberapa teknologi dasar web digunakan untuk membangun simulasi sistem saklar digital. Berikut adalah teknologi yang berperan utama:

HTML & CSS

Untuk membuat tampilan web dan simulasi visual saklar.

JavaScript

Mengontrol perubahan status lampu melalui event tombol.

GIF Animation

Memberi efek visual seolah lampu benar-benar menyala dan mati.

Rencana Pengembangan


Ke depan, sistem SadewaTech akan dikembangkan menjadi saklar otomatis yang bisa terhubung dengan jaringan Wi-Fi, sehingga dapat dikontrol dari smartphone atau komputer melalui aplikasi berbasis IoT.
